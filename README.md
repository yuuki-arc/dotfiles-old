## インストール
### dotfiles
1. cd $HOME
1. git clone
1. sh -x ./dotfiles/setup.sh

### tmux 
1. brew tap homebrew/dupes
1. brew install homebrew/dupes/grep
1. fc-cache -f

### vim - NeoBundleInstall
1. git clone https://github.com/Shougo/neobundle.vim.git ~/.vim/bundle/neobundle.vim
1. :NeoBundleInstall
